## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/Diyan-Welikanna/my-portfolio.git
cd my-portfolio
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Update Personal Information

1. Edit the hero section in \`components/Hero.tsx\`
2. Update the about section in \`components/About.tsx\`
3. Add your projects in \`components/Projects.tsx\`
4. Update your skills in \`components/Skills.tsx\`
5. Update contact information in \`components/Contact.tsx\` and \`components/Footer.tsx\`

### Change Colors

Edit the \`tailwind.config.ts\` file to customize the color scheme.
